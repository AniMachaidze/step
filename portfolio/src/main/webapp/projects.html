Suchmaschine - Search Engine 03/2020-05/2020

CIS 455 Internet and Web Systems Final Project

Collaborators: Michael Colombo, Hanbang Wang, Chris Williams

Suchmascine is a google-style search engine. The search engine has 5 major components. The Crawler, which crawles the web and puts crawled pages in the database; Pagerank Algorithm, which calculates pagerank for each crawled page; Indexer, which runs TF-IDF algorithm and calculates vectors; User interface, which accept user query and gets pages based on the calculated rankings.

My contribution:

  • Implemented Mercator-style distributed Web Crawler, which crawls ~3 pages/second.

  • Parallelized the processes using Apache Spark.

  • Contributed toward int egration of the project and summarizing in the paper.

Covid-19 Spread Simulation 05/2020

NETS 150 Network and Social Systems Engineering Final Project

Collaborator: Meri Kavtelishvili

In our project, we tried to model the spread of the virus using the SIR model. SIR model considers different parameters: graph structure (node isolations, community clusters), probability of infecting person infect its specific neighbour, number of iterations for which person stays infected. Based on these parameters, we modeled the problem and experimented to prove our hypothesis about the virus.

My contribution:

  • Implemented an application that generates graph based on several variables such as number of neighbours, probability for each of other nodes to be connected.

  • Implemented an application that simulates spread of a virus.

  • Created an empirical analysis based on different variables and observing spread simulation.
